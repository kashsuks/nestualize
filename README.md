# nestualize
Visualize Hack Club Nest

## Local Development

- Clone the repository locally:
`git clone https://github.com/kashsuks/nestualize"`

- Change directory (cd) into the cloned directory
- Install dependencies
`npm install`
- Run locally
`npm start`
